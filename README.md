# HR Analytics: Employee Attrition Case Study

## Project Overview
Proyek ini bertujuan untuk mengidentifikasi faktor-faktor utama yang menyebabkan atrisi karyawan menggunakan Python. Sebagai profesional HR, analisis ini membantu manajemen mengambil keputusan berbasis data untuk meningkatkan retensi.

## Tahapan Proyek
1. **Data Cleaning:** Penanganan missing values dan audit kualitas data.
2. **Exploratory Data Analysis (EDA):** Visualisasi korelasi antara kepuasan kerja, gaji, dan beban kerja.
3. **Executive Summary:** Pembuatan laporan otomatis menggunakan library `python-pptx`.

## Key Insights
* Karyawan di posisi teknis memiliki risiko atrisi tertinggi.
* Gaji tinggi tidak menjamin loyalitas jika skor kepuasan kerja rendah.
* Jarak rumah yang jauh berkorelasi signifikan dengan peningkatan risiko atrisi.

## Tools Used
* **Python** (Pandas, Seaborn, Matplotlib)
* **Jupyter Notebook**
